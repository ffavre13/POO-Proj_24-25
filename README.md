## Controls
**Hero :** 
- `WASD` to move
- `↑ ← → ↓` to change the direction of fire
- `space` to shoot
- `R` to restart the game

## Launch the project
**Using IntelliJ Community :**
1. Clone the repository
2. Set up your JDK and Scala SDK
4. Launch the `src/Game.scala` file.
5. Enjoy ! :smile:
